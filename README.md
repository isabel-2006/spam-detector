# 📩 AI Spam Detector

An AI-powered Spam Detection Web Application built using **Python, Flask, Machine Learning, and NLP**.  
The application analyzes SMS/text messages and predicts whether the message is **Spam** or **Not Spam** with a confidence score.

---

# 🚀 Features

✅ Detects Spam and Non-Spam messages  
✅ Machine Learning based prediction  
✅ NLP text processing using TF-IDF  
✅ Confidence percentage display  
✅ Dynamic color-changing UI  
✅ Simple and responsive web interface  
✅ Built with Flask backend

---

# 🧠 Technologies Used

- **Python**
- **Flask**
- **Scikit-learn**
- **Machine Learning**
- **Natural Language Processing (NLP)**
- **TF-IDF Vectorizer**
- **Naive Bayes Classifier**
